# Decision Log

Architectural decisions for this system. Each decision has a dedicated ADR file.

## Index

| ID | Summary | Status | Date |
|----|---------|--------|------|
| ADR-001 | [Decision title](./adr/ADR-001.md) | accepted | YYYY-MM-DD |

## Status Values

- **proposed** - Under discussion
- **accepted** - Approved and in effect
- **deprecated** - No longer applies
- **superseded** - Replaced by another ADR

## Creating New Decisions

1. Copy `adr.template.md` to `./adr/ADR-NNN.md`
2. Fill in the template
3. Add entry to index above
4. Update status as decision progresses
